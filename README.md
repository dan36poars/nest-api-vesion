## Description

Initial API learning

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Commands Uses

```bash
# generate a controller
$ nest g controller name_controller

# generate a service
$ nest g service name_service

# generate a resource(CRUD API)
$ nest g resource name_resource
```
